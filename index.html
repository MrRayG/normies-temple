THE NORMIES PRIMER

New to the ecosystem? Start here. Everything you need to understand what makes Normies unlike anything else in Web3.

01

WHAT IS A NORMIE?

Each Normie is a 40×40 monochrome bitmap — 1,600 pixels. Generated by AI at mint and permanently stored inside Ethereum smart contracts as SVG data. No IPFS. No servers. If Ethereum exists, so do Normies.

READ DOCS →
02

HOW DOES BURNING WORK?

You burn one Normie to give another Action Points. The burned token is gone forever — its pixels permanently contributed to the system. Action Points = creative fuel. More burns = more edits possible = more artistic freedom.

BURN DOCS →
03

THE CANVAS MECHANIC

Normie Canvas lets you spend Action Points to toggle individual pixels on your Normie. Each pixel change is an on-chain transaction. The Canvas layer is an XOR overlay — it shows only what changed. Traits on marketplaces don't auto-update, but the on-chain data does.

CANVAS DOCS →
04

LEVELS & ACTION POINTS

Every Normie starts at Level 1. As it receives burns from other Normies, its level increases. Higher levels unlock more pixel edits per action point and represent how much collective creative energy has been poured into that single token.

LEVEL SYSTEM →
05

CC0 & INTEROPERABILITY

Normies are CC0 — no rights reserved. Anyone can use, build on, or remix the art without permission. XNORMIES, Normies Run, Normies 3D — all community-built derivatives. The IP belongs to everyone who holds it.

LEARN MORE →
06

THE 100 & LEGENDARY CANVASES

Some Normies transcend PFPs. Skelemoon — born from 50 burns and the brush of A.C.K. — was the first Legendary Canvas. #The100 is a community-proposed recognition structure for the 100 greatest Normie canvases ever created. History forged on-chain.
